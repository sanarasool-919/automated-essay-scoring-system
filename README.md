# Automated Essay Scoring System

This project uses Machine Learning and NLP techniques to automatically score essays based on their text content.

## Features
- TF-IDF feature extraction
- Logistic Regression model
- SVM classification
- Evaluation using Accuracy, F1-score, and Cohen Kappa

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Models
- Logistic Regression (Baseline)
- Linear SVM (Optimized)
